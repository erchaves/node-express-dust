## Dev notes

Install the dependencies:
```bash
npm install
```
Running locally:
```bash
npm run dev
# or just run 'gulp' if you have it installed globally.
# The app will be served at `localhost:3000`
```

Running on Prod:
```bash
npm start
# npm prestart will run gulp build before it starts the server
```
